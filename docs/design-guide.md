# 농사일지 — 디자인 가이드

- 버전: v0.1
- 최종 수정: 2026-07-13

전체적으로 파란 계열, Noto Sans KR 고딕체를 기본으로 하는 실용적인 디자인. 화려한 랜딩페이지가 아니라 매일 쓰는 기록 도구이므로, 장식보다 가독성과 일관성을 우선한다.

## 컬러

### Primary (Blue)

| 토큰 | HEX | 용도 |
|---|---|---|
| `primary-50` | `#EFF5FC` | 선택된 칩/버튼의 옅은 배경 |
| `primary-100` | `#DCEAF9` | 호버/보더 배경 |
| `primary-300` | `#8FBBEA` | 보조 보더 |
| `primary-500` | `#3379C9` | 링크, 보조 강조 |
| `primary-600` | `#235FA8` | 기본 버튼/주요 강조 (기본값) |
| `primary-700` | `#1C4B86` | 버튼 hover/active |

### Neutral (Blue-tinted Gray)

| 토큰 | HEX | 용도 |
|---|---|---|
| `neutral-50` | `#F7F9FB` | 페이지 배경 |
| `neutral-100` | `#EEF2F6` | 카드/섹션 배경 |
| `neutral-200` | `#DCE3EA` | 보더 |
| `neutral-400` | `#97A6B5` | placeholder, 비활성 텍스트 |
| `neutral-500` | `#6B7A89` | 보조 텍스트 |
| `neutral-700` | `#3E4C5A` | 본문 텍스트 |
| `neutral-900` | `#1B2530` | 제목 |

### Semantic (태그 전용, Primary와 별도)

| 토큰 | HEX | 용도 |
|---|---|---|
| `sow` | `#235FA8` (primary-600 재사용) | 파종 태그 — 손글씨 일지의 파란 동그라미와 동일한 의미 |
| `harvest` | `#C1462F` | 수확 태그 — 손글씨 일지의 빨간 동그라미와 동일한 의미 |
| `cost` | `#B8791A` | 비용 표시 (달력 점 등) |

파종 태그가 Primary와 같은 파란색을 쓰는 것은 의도된 선택이다 — 사용자가 실제 손으로 쓰던 일지에서도 파종을 파란색으로 표시했기 때문에, 앱의 기본 색과 자연스럽게 겹치도록 둔다.

## 타이포그래피

- 서체: **Noto Sans KR** (`next/font/google`로 자체 호스팅, 외부 CDN 요청 없음)
- 굵기: 400(본문), 500(라벨/버튼), 700(제목)
- 본문 크기: 14px(0.875rem) 기본, 제목은 18–20px

## 아이콘

텍스트 라벨을 아이콘으로 대체하되, 의미가 즉시 파악되도록 아이콘 + 짧은 텍스트를 함께 쓴다 (아이콘만 쓰면 오히려 헷갈릴 수 있음). `lucide-react` 사용.

## 컴포넌트 패턴

- **칩(chip)**: 미선택 시 `neutral-200` 보더 + `neutral-500` 텍스트, 선택 시 `primary-600` 배경 + 흰 텍스트. 둥근 pill 형태 유지.
- **기본 버튼**: `primary-600` 배경, hover 시 `primary-700`.
- **보조/고스트 버튼** (줄 추가, 비용 추가 등): 점선 보더, `neutral-400` 텍스트, 배경 없음.
- **카드/섹션**: `neutral-100` 배경 또는 `neutral-200` 보더, 라운드 8px.
